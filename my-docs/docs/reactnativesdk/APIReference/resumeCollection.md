# Resume Collection

The `resumeCollection` method restarts event collection if it was previously paused using the `pauseCollection` method.

## Syntax

```javascript
resumeCollection()
```

## Usage Example

```javascript
import { resumeCollection } from 'zeo-collect';

// Resume data collection
resumeCollection();
```
