# Pause Collection

The `pauseCollection` method temporarily suspends the package from collecting events for user actions.

## Syntax

```javascript
pauseCollection()
```

## Usage Example

```javascript
import { pauseCollection } from 'zeo-collect';

// Pause data collection
pauseCollection();
```
